Access Notes

Common cases that block login.

I paid but cannot access yet.

Make sure you login with the same email used at checkout and wait a minute for Stripe status to sync.

I did not receive my verification code.

Check spam/promotions folders, then request a new code after a short wait.

Can free users access the app?

The public site is open to everyone, but workspace and member booking are paid-access only.
